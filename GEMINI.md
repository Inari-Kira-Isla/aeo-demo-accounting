# aeo-demo-accounting

## Overview
A static HTML template designed for the accounting industry. It serves as a demonstration of AI-Engine Optimization (AEO) best practices, utilizing Schema.org, `llms.txt`, and semantic FAQ markup to enhance visibility for AI agents and crawlers.

## Tech Stack
*   HTML5
*   Schema.org (JSON-LD)
*   llms.txt
*   Semantic HTML (Accessibility)

## Architecture
*   **Root**: Main entry point (`index.html`) and industry-specific pages.
*   **Structure**: Flat HTML structure prioritizing semantic tags (`<article>`, `<section>`) and structured data injection.
*   **AEO Components**: Dedicated sections for FAQs and Schema implementation.

## Commands
*   **Preview**: Open `index.html` in a browser or serve locally (e.g., `python3 -m http.server`).
*   **Deploy**: Push to a GitHub repository to enable GitHub Pages for public access.

## Coding Style
*   **Semantic Markup**: Strict use of HTML5 tags for meaning.
*   **Schema Validation**: Ensure JSON-LD is valid and contextually accurate.
*   **AI-Friendly**: Prioritize clear content structure and readability for LLM parsing.

## Important Rules
*   **Do Not Remove**: Keep the `llms.txt` link and JSON-LD scripts intact; they are core to the demo's purpose.
*   **Content Integrity**: Maintain high text quality; AI agents rely heavily on context and clear data relationships.
